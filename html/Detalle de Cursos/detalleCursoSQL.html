Curso de SQL

Tiempo de Dedicacion Necesario: 35hs

Este curso enseña los fundamentos del lenguaje SQL, utilizado para la gestión y consulta de bases de datos relacionales. Los estudiantes aprenderán a crear, modificar y consultar tablas, además de realizar operaciones básicas y avanzadas para trabajar con información de manera eficiente.

Requisitos Previos: Excel (Basico)

$40.000

Contenidos Por Clase

INTRODUCCIÓN
  • ¿Qué es SQL y para qué sirve?
  • Bases de datos relacionales: conceptos básicos.
  • Instalación y uso de un gestor de bases de datos (MySQL / PostgreSQL).
UNIDAD 1
  • Creación y gestión de bases de datos.
  • Tablas: creación, modificación y eliminación.
  • Tipos de datos en SQL.
  • Consultas básicas con SELECT y filtros con WHERE.
UNIDAD 2
  • Operadores y funciones en SQL (agregación y matemáticas).
  • Ordenar y agrupar resultados (ORDER BY, GROUP BY, HAVING).
  • Relaciones entre tablas: claves primarias y foráneas.
  • Joins: INNER, LEFT, RIGHT y FULL JOIN.
